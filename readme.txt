Các phần mềm cần cài:
	- Nodejs
	- Mongodb
	- Mongodb compass
	- Postman
Tạo database:
	- Chạy mongodb compass
	- Tạo database tên là "mydb"
	- Tạo các collection: categories, list_products, products, users
	- Chọn từng collection, chọn Collection (trên thanh menu), chọn Import data. Chọn file JSON tương ứng trong thư mục mydb của project này.
Chạy:
	- Chạy app.js (bằng lệnh node app.js). Server sẽ được chạy trên localhost:80
	- Dùng Postman để gửi các request đến server
